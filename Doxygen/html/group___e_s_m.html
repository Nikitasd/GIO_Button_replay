Подробное описание

Error Signaling Module.

The ESM module aggregates device errors and provides internal and external error response based on error severity.

◆ esmGroup1Notification()

void esmGroup1Notification ( uint32  channel)

Interrupt callback.

Аргументы
[in]channel- Group 1 channel

This is a callback that is provided by the application and is called upon an interrupt. The parameter passed to the callback is group 1 channel caused the interrupt.

Заметки
Callback parameter channel is not a masked value. Do not use the macros esmCHANNELx for comparison.
