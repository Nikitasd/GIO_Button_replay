Подробное описание

RTI Register Frame Definition.

This type is used to access the RTI Registers.

Поля

◆ CAFRCx

uint32 CAFRCx

0x0020,0x0040: Capture Free Running Counter x Register

◆ CAPCTRL

uint32 CAPCTRL

0x0008: Capture Control Register

◆ CAUCx

uint32 CAUCx

0x0024,0x0044: Capture Up Counter x Register

◆ CLEARINTENA

uint32 CLEARINTENA

0x0084: Clear/Status Interrupt Register

◆ 

struct { ... } CMP[4U]

Compare x selection:

  • 0: Compare 0
  • 1: Compare 1
  • 2: Compare 2
  • 3: Compare 3

◆ 

struct { ... } CNT[2U]

Counter x selection:

  • 0: Counter 0
  • 1: Counter 1

◆ COMP0CLR

uint32 COMP0CLR

0x00B0: RTI Compare 0 Clear Register

uint32 COMP1CLR

0x00B4: RTI Compare 1 Clear Register

uint32 COMP3CLR

uint32 COMPCTRL
